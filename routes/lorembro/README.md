### lorembro

CORS friendly HTTP GET API for retrieving dummy data.

API Endpoint: **api/lorembro**

Accepted query string key:value pairs.

* **data**
    * {String} Which of the data samples to retrieve.
    * Accepted Values:
        * accesslog
        * html
        * json
        * longtext
        * **text (default)**
        * wordlist
        * xml
        * yaml
